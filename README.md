Task 1: Introduction

Task 3: Initialization and Game Loop
Task 4: Images, Blit and Flip
Task 5: Animal Class
Task 6: Displaying Animal Images
Task 7: Handling Events
Task 8: Displaying Recent Images
Task 9: Finishing Touches
